# doc-builder
An simple runner of rst2html5 for batch converting of rst files

`rst2html5` must be installed:
$ rst2html5-tools

$ git clone https://github.com/oriontvv/doc-builder doc
$ ./builder.py
$ exa --tree
doc
├── bld
│  └── api
│     └── module.html
├── build.py
└── src
   └── api
      └── module.rst
